## Getting Started

Requirement:

- NodeJS +

Install dependencies:

```
npm install
```
## Running production server

```
npm start
```

Visit [http://localhost:8000/](http://localhost:8000/).


## Running development server

Run webpack dev server (for assets):

```
npm run start:dev
```

Visit [http://localhost:8000/](http://localhost:8000/).


### Testing

To run tests, use:

```
npm test
```
